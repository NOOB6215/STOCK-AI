# 🤖 주식 AI 리포트

한국 주식 시장 AI 분석 도구 - Streamlit 기반 웹앱

## ✨ 주요 기능

- 📈 **실시간 주가 차트**: 1개월~3년, MA/RSI 지표
- 📰 **뉴스 AI 감성 분석**: 한국어 금융 특화 (KR-FinBert)
- 📋 **DART 공시**: 금융감독원 공식 공시 자동 수집
- 🤖 **XGBoost AI 예측**: 단기 상승 확률 계산
- 💎 **장기 가치 평가**: PER/PBR/ROE/배당수익률
- 🎓 **Claude 전문가 리포트**: Claude Sonnet 4.5 기반 자연어 분석

## 🚀 지원 종목

삼성전자, SK하이닉스, LG전자, 네이버, 카카오

## ⚠️ 주의사항

이 앱은 **투자 참고용**입니다. 실제 매매는 본인 판단과 책임입니다.

## 🛠️ 기술 스택

- **Frontend**: Streamlit, Plotly
- **ML**: XGBoost, PyTorch, Transformers
- **AI**: Claude Sonnet 4.5 (Anthropic)
- **Data**: Yahoo Finance, Google News, DART API
